변수 선언방법 3가지
var, let, const

var 옛날 브라우저를 지원해야 할 경우
let , const 2015년 es6 에서 추가 ==> 블럭 

const 는 값을 변경하지 않아야 하는 경우(상수)
배열, 객체에서 유형을 변경하지 않을 경우


세가지 경고 출력 방법
-alert   경고창
 ex)alert("경고!!!");
-prompt  데이터값 입력창
 ex)let age = prompt("당신의 나이를 입력하세요.");
-confirm  bool값 창
 ex)let y = confirm("당신은 18세 이상입니까?");
