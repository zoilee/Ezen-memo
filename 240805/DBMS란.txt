DBMS란?

Data base Management System

-데이터베이스(db)를 관리(ms)하는 시스템
-db: 테이블들이 모여 이루는 단위

-데이터베이스 유지보수
-db:테이블들이 모여 이루는 단위
 

- 데이터베이스 유지보수 (만들고, 수정, 삭제, 읽고)
 - CRUD (Create, Retrieve, Update, Delete)

- 대량의 데이터를 처리하는 시스템
- 다양한 자료구조와 검색구조(소팅, 인덱싱 ...)사용해 빠른 검색 가능
- 대부분의 시스템은 R>>>>>>>CUD의 빈도수
- 검색엔진 최적화

Index

인덱스 종류
 - 이진검색 (Binary Search) - 최대 log₂ (N)번 내에 검색 가능
 - B-TREE 계열 - 상용 DBMS의 가장 일반적인 검색
   * 최대 log3 (N)
   * 사용 DBMS의 가장 일반적인 검색방법

   데이터 추가/ 삭제/ 수정 할 때마다 정렬/인덱스의 업데이트가 일어난다.

DBMS
 - 계층형 데이터베이스
 - 네트워크형 데이터베이스
 - 관계형 데이터베이스 (RDBMS) - db의 전용언어 sql 언어 ***
 - 객체지향 데이터베이스
 - 객체관계형 데이터베이스
 - NoSQL (Not Only SQL) - 몽고DB  ***
